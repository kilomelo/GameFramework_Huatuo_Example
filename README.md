# GameFramework_Huatuo_Example
Huatuo接入GameFramework的AB工作流，并完善了相关编辑器工具。

支持基于构建目标的Huatuo开关配置  
支持在GameFramework的ResourceBuild流程中完成Hotfix Assembly的编译、拷贝、打包  
支持编辑器中一键设置是否使用本地Assembly进行调试
